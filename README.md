# TechStudio3D

TechStudio3D is a modern web application designed to showcase 3D services and products. It provides a clean and intuitive interface for exploring products, viewing detailed specifications, and quickly contacting the team via WhatsApp or a request form.

---

## 🚀 Features

* 📦 Product catalog with structured data
* 🔍 Detailed product pages
* 🖼 Image gallery for each product
* 📋 Technical specifications and descriptions
* 💬 Instant communication via WhatsApp
* 📨 Request Quote modal form
* 🎨 Modern UI with glassmorphism and glow effects
* ♻️ Reusable component-based architecture

---

## 🛠 Tech Stack

* **React**
* **React Router**
* **SCSS Modules**
* **Feature-Sliced Design (FSD)**

---

## 📂 Project Structure

```bash
src/
 ├── app/           # global styles and configuration
 ├── entities/      # business entities (Product, etc.)
 ├── features/      # feature modules (RequestQuote)
 ├── shared/        # reusable components
 ├── widgets/       # complex UI blocks (Gallery, etc.)
 └── pages/         # application pages
```

---

## ⚙️ Installation & Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```
